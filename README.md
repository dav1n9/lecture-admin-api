# lecture-admin-api
강의 사이트의 백오피스(admin) 서버 api 입니다.

Spring Security 를 적용하지 않고, JWT 를 사용하여 인증/인가를 구현했습니다.

## ✅ 주요 기능
❗관리자 가입과 로그인을 제외한 모든 기능은 관리자만 접근 가능합니다.

1. 관리자 회원 가입, 로그인
2. 권한에 따른 강사 및 강의 등록, 수정, 삭제, 조회
3. 선택한 강사가 촬영한 강의 목록 조회 기능
4. 카테고리별 강의 목록 조회 기능
